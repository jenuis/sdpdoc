==========================================
The SDP User Guide!
==========================================

The synthetic diagnostic platform (SDP) is the platform to host the synthetic diagnostic models, the integrated data analysis (IDA) tools, and the data visualization tools, etc, for experimental and simulation studies. In this guide, the basic information about SDP and the user tools will be briefly introduced.
